## Документация библиотек Python

### Статистические тесты
| Библиотека | Ссылка | Когда использовать |
|------------|--------|-------------------|
| **SciPy Stats** | [docs.scipy.org](https://docs.scipy.org/doc/scipy/reference/stats.html) | Основные статтесты, базовый выбор |
| **Pingouin** | [pingouin-stats.org](https://pingouin-stats.org/) | Удобный интерфейс, больше статистик в выводе |
| **Statsmodels** | [statsmodels.org](https://www.statsmodels.org/stable/) | Регрессии, ANOVA, коррекция множественных сравнений |

### Конкретные методы — быстрые ссылки
| Метод | Ссылка на API |
|-------|--------------|
| `pingouin.anova` | [anova](https://pingouin-stats.org/build/html/generated/pingouin.anova.html) |
| `pingouin.ttest` | [ttest](https://pingouin-stats.org/build/html/generated/pingouin.ttest.html) |
| `pingouin.mwu` | [Mann-Whitney U](https://pingouin-stats.org/build/html/generated/pingouin.mwu.html) |
| `pingouin.normality` | [Тест нормальности](https://pingouin-stats.org/build/html/generated/pingouin.normality.html) |
| `scipy.stats.bootstrap` | [Bootstrap](https://docs.scipy.org/doc/scipy/reference/generated/scipy.stats.bootstrap.html) |
| `statsmodels multitest` | [Поправки на множественность](https://www.statsmodels.org/stable/generated/statsmodels.stats.multitest.multipletests.html) |
